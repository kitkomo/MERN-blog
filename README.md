# MERN-blog

## Frontend

### Stack:
- React.js
- Redux Toolkit
- React Router
- React Hook Form
- React Markdown
- Material UI
- Axios

## Backend
To start working with a server provide a `.env` file with the following variables:
- `PORT`
- `JWT_SECRET`
- `MONGO_DB_URI`

Scripts:
- `npm run dev` - start server via nodemon in development mode
- `npm run start` - start server via node.js

### Stack:
- Node.js
- Express (+ express-validator)
- MongoDB
- Mongoose
- JSON Web Token
- Multer
- Bcrypt
